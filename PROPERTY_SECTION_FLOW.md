## Component States

| State | Shows | Triggers |
|-------|-------|----------|
| **Search** | Input + Google autocomplete dropdown | Initial state, or after "Change" |
| **Loading** | Skeleton + spinner + "Fetching..." text | After address selected |
| **Unit Picker** | Building address + list of units | API returns multiple matches |
| **Complete** | Green success card + legal description | Single property found or unit selected |
| **Error** | Red alert with message | API error or not found |

---

## Key UX Decisions

1. **Google autocomplete** appears as user types (after 3 chars)
2. **Loading skeleton** replaces the entire form (not just a spinner)
3. **Unit picker** is a separate state, not a modal
4. **Auto-advance** to Grantor section after property confirmed
5. **"Change" button** resets back to search state
6. **Legal description** shown in scrollable box (max 128px height)

## API Response Handling

```typescript
// Expected API response shapes:

// Success - Single Property
{
  status: 'success',
  data: {
    address: '123 Main St',
    city: 'Los Angeles',
    county: 'Los Angeles',
    state: 'California',
    zip_code: '90012',
    apn: '5432-001-012',
    legal_description: 'LOT 12 OF TRACT...',
    primary_owner: { full_name: 'JOHN SMITH' },
    secondary_owner: { full_name: 'JANE SMITH' }
  }
}

// Multiple Units (Condo)
{
  status: 'multiple_matches',
  matches: [
    { address: '123 Main St #101', unit: 'Unit 101', apn: '5432-001-012' },
    { address: '123 Main St #102', unit: 'Unit 102', apn: '5432-001-013' },
    { address: '123 Main St #103', unit: 'Unit 103', apn: '5432-001-014' },
  ]
}

// Error
{
  status: 'error',
  message: 'Property not found. Please check the address.'
}
```
